# House-Prediction-Model
This project is a Machine Learning-based House Price Prediction system that estimates property prices based on various housing features such as area, number of rooms, location factors, and other relevant attributes.

* Project Overview

Built using Python and Jupyter Notebook

Performs data preprocessing, visualization, and feature engineering

Trains and evaluates regression models for accurate price prediction

Uses real-world housing dataset for model training and testing

* Technologies Used

Python

Pandas

NumPy

Matplotlib / Seaborn

Scikit-learn

Jupyter Notebook

* Key Features

Data cleaning and preprocessing

Exploratory Data Analysis (EDA)

Model training and evaluation

Performance metrics analysis

Predicts house prices based on user input features
